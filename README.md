# TiddlyWiki Plugins web pages

gh-pages for [sycom's TiddlyWiki plugins](http://sycom.github.io/TiddlyWiki-Plugins). Latest version will rather be on [gitlab][gitlab-io]. View a demo TiddlyWiki with

* [leaflet plugin][l-m]
* [Google analytics plugin][g-a]
* [feather icons plugin][f-i]
* [atom feed plugin][a-f]
* [i18n macro][i18n]

Also find some support function for plugins as :

* [draw for leaflet plugin](https://sycom.github.io/TiddlyWiki-Plugins/leafletDraw.html) which help you design objects for your wiki while draw is not implemented in the plugin.
